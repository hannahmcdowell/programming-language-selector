, get ready to learn Python!

Python is an interpreted high-level general-purpose programming language
Why Python?
  • Python works on different platforms.
  • Python has simple syntax, similar to the English language.
  • Python runs on an interpreter system, meaning that code can be executed as soon as it is written.
  • It's super popular!

, get ready to learn JavaScript!

JavaScript (JS) is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions
Why JavaScript?
  • JavaScript is the world's most popular programming language.
  • JavaScript is the programming language of the web.
  • JavaScript is easy to learn.

, get ready to learn Java!

Java is a high-level, class-based, object-oriented programming language
Why Java?
  • Java is the most in-demand programming language.
  • Java has an abundant API.
  • Java is platform independent (it's known as the "write once, run anywhere" language).

, get ready to learn C!

C is the most commonly used programming language for writing operating systems
Why C?
  • C will help you understand the fundamentals of computer theories.
  • C can be used for writing operating systems as well as doing application level programming.
  • Since C has fewer libraries, it will help you learn how to write a lot of things from scratch.
